# Gavia.Store.fn
オブジェクトストアへのメソッド群はGavia.Store.fnに定義されています。  
また、Gavia.Store.fnへ新たにユーザー定義メソッドを作ることも出来ます。  

また、ほぼ全てのメソッドはjQueryDeferredを返します。
